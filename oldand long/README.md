# Portfolio Website

This folder contains a personal portfolio website that is ready for GitHub Pages hosting.

## Files

- `index.html` - main page layout
- `styles.css` - full website styling
- `script.js` - renders your content from JSON
- `data/portfolio.json` - main content file to edit later
- `assets/ShubhamPatel_Resume_2025.pdf` - your linked CV
- `assets/FINAL_18BME099_CP_REPORT.pdf` - your project report

## What to edit next

Replace the placeholders inside `data/portfolio.json` for:
- email
- GitHub
- LinkedIn
- full about text
- exact project titles and descriptions
- education and experience timeline

## GitHub Pages

1. Create a GitHub repository.
2. Upload everything from this `protfolio` folder.
3. In GitHub open `Settings` > `Pages`.
4. Choose `Deploy from a branch`.
5. Select branch `main` and folder `/ (root)`.
6. Save.

Your site will be available at:

`https://YOUR-USERNAME.github.io/REPOSITORY-NAME/`
